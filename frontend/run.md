# Run Frontend

Follow these steps to run the React frontend.

## Prerequisites
1. Make sure you have **NodeJS 18+** installed.

---

## Run the React Frontend

1. Open a terminal window and navigate to the `frontend` folder:
   ```powershell
   cd "c:\Users\KIIT\Desktop\New folder (3)\One-Click-Automation\frontend"
   ```
2. Install dependencies (if you haven't already):
   ```powershell
   npm install
   ```
3. Run the Vite development server:
   ```powershell
   npm run dev
   ```
4. Open your browser and navigate to the URL shown in the console (usually [http://localhost:5173](http://localhost:5173)).
